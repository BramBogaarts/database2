# database2
database
Een PHP-gebaseerde webapplicatie voor het beheren van een games database met gebruikersauthenticatie en rol-gebaseerde toegangscontrole.

Technische vereisten: PHP: 8.1.0 of hoger Mysql opdrachtprompt (om composers te installeren). MAMP (om lokaal te werken).
installatie: git clone: https://github.com/BramBogaarts/database2.git cd "de locatie waar je de mappen wilt hebben"
gebruik: log in met je account als je er al een hebt en anders maak je er een aan.
als je bent ingelogd kun je de gamelijst bekijken.
